Política de Privacidad - Quedada

Última actualización: 14 de abril de 2025

Quedada organiza eventos y conecta usuarios vía Firestore, Supabase, EmailJS y AdMob, cumpliendo con el RGPD.

1. Datos que recopilamos

  • Nombre: Para perfil, eventos, chats (Firestore).
  • Correo electrónico: Registro, contacto (Firestore, EmailJS).
  • ID de usuario: Autenticación (Firestore).
  • Mensajes: Chats en la app (Firestore).
  • Fotos: Imágenes para eventos (Supabase).
  • Eventos de calendario: Agendas (Firestore).
  • Contenido generado: Grupos, eventos, fotos (Firestore, Supabase).
  • ID de publicidad: Anuncios (AdMob).
  • Interacciones: Analíticas (AdMob).
  • Informes de fallos: Mejora de la app (Capacitor/AdMob).

2. Cómo usamos tus datos

  • Nombre: Mostrar en eventos, chats, autenticación.
  • Gestionar cuentas, eventos, agendas (Firestore).
  • Subir fotos (Supabase).
  • Procesar consultas (EmailJS).
  • Mostrar anuncios (AdMob).

3. Compartir datos

  • Firestore: Almacena nombres, mensajes, eventos.
  • Supabase: Almacena fotos.
  • EmailJS: Correos de contacto.
  • AdMob: ID de publicidad (política).
  • Nombres y fotos visibles en la app (funcionalidad).
  • No compartimos datos personales con terceros sin consentimiento.

4. Seguridad

Usamos HTTPS y medidas de seguridad.
